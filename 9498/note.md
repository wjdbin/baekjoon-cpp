# 백준 9498번: 시험 성적 C++ 핵심 정리

## 1. 문제 요약

시험 점수(0~100점)를 정수로 입력받아, 점수대에 맞는 등급(A, B, C, D, F)을 출력하는 문제입니다.

* **90 ~ 100점:** A
* **80 ~ 89점:** B
* **70 ~ 79점:** C
* **60 ~ 69점:** D
* **나머지:** F

## 2. C++ 핵심 문법 및 로직

이 문제를 해결하기 위해 C++의 두 가지 주요 기능이 사용됩니다.

### 📌 1. 표준 입출력 (`<iostream>`)

* **입력:** `std::cin`을 사용하여 사용자로부터 점수를 정수형 변수(예: `int score`)에 저장합니다.
* **출력:** `std::cout`을 사용하여 결과 등급(문자열)을 화면에 출력합니다.

### 📌 2. 조건문 (`if-else if-else`)

이 문제의 핵심입니다. 점수 범위에 따라 다른 출력을 해야 하므로, 다중 조건 분기가 필요합니다.

**중요한 점:** `if-else if` 구조를 사용하면, **가장 높은 점수대부터 순서대로 검사**하는 것이 가장 효율적입니다.

* 만약 `score >= 90`이 참이면, `A`를 출력하고 나머지 `else if`와 `else`는 모두 건너뜁니다.
* 만약 `score`가 85라면, 첫 번째 `if (score >= 90)`은 거짓이 되고, 두 번째 `else if (score >= 80)`이 참이 되어 `B`를 출력하고 나머지를 건너뜁니다.
