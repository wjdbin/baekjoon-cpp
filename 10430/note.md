# 📘 백준 10430번 오답노트

## 문제
세 수 `A, B, C`가 주어졌을 때, 다음 네 가지 값을 출력하는 문제:

1. `(A + B) % C`
2. `((A % C) + (B % C)) % C`
3. `(A * B) % C`
4. `((A % C) * (B % C)) % C`

---

## 핵심 개념
- **모듈러 연산(%)의 분배 법칙**
  - 덧셈: `(A + B) % C = ((A % C) + (B % C)) % C`
  - 곱셈: `(A * B) % C = ((A % C) * (B % C)) % C`
- 주어진 수식들이 모두 동일하게 성립함을 확인하는 문제.

---

## 주의할 점
1. 나눗셈이 아니라 **나머지 연산 `%`**을 써야 함.  
2. 코드 에러가 난다면:
   - 확장자가 `.cpp`인지 확인  
   - C++ 컴파일러(g++) 사용 필요
