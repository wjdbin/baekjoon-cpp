# 백준 1330: 두 수 비교하기 (C++) 핵심 요약

## 1. 문제
두 정수 A, B를 입력받아 크기를 비교하여 `>`, `<`, `==` 중 하나를 출력합니다.

## 2. 핵심 문법: `if-else if-else`
* `if (A > B)`: A가 B보다 클 때
* `else if (A < B)`: (A가 B보다 크지 않고) A가 B보다 작을 때
* `else`: 둘 다 아닐 때 (즉, A == B일 때)

## 3. 치명적 실수: '=' 와 '==' 혼동
* `if (A == B)`: **(O) 비교:** A와 B가 같다.
* `if (A = B)`: **(X) 대입:** A에 B의 값을 넣는다. (의도와 다르게 작동)
