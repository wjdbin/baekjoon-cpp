# [오답노트] 백준 14681번: 사분면 고르기

## 1. 문제 분석
* **문제:** 주어진 좌표 (x, y)가 몇 사분면에 속하는지 출력하기
* **핵심:** 각 사분면의 x, y 부호 조건 파악하기

## 2. 틀린 이유 (Bug Report)

### ① 문법 오류 (Syntax Error)
* **작성 코드:** `cin >> x >> "\n";`
* **원인:** `cin` 뒤에는 값을 저장할 **변수**가 와야 함. `"\n"`(문자열)은 올 수 없음.
* **해결:** `cin`은 공백과 줄바꿈을 자동으로 무시하므로 `cin >> x;` 만 작성하면 됨.

### ② 논리 오류 (Logic Error)
* **증상:** 2사분면과 4사분면의 결과가 반대로 나옴.
* **원인:** 사분면의 좌표 부호 조건을 헷갈림.
    * (작성한 것) x > 0, y < 0 일 때 2사분면이라고 판단함 (틀림)
    * (실제) x > 0, y < 0 은 **4사분면**임.
