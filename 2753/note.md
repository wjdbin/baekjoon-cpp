## 📅 백준 2753번: 윤년

---

### 1. 문제 개요
연도가 주어졌을 때, 그 해가 **윤년**인지 아닌지를 판별하는 문제입니다.

---

### 2. 입력
* 첫째 줄에 연도를 나타내는 정수 1개가 주어집니다.
* (범위: $1 \le \text{연도} \le 4000$)

---

### 3. 출력
* 입력된 연도가 윤년이면 **1**을 출력합니다.
* 윤년이 아니면 **0**을 출력합니다.

---

### 4. 핵심 로직 (윤년의 조건)
컴퓨터 프로그래밍에서 윤년은 다음 두 조건 중 하나를 만족해야 합니다.

* **조건 1:** 연도가 4의 배수이면서 ( `year % 4 == 0` ) **동시에** 100의 배수가 아니어야 합니다. ( `year % 100 != 0` )
* **조건 2:** 연도가 400의 배수여야 합니다. ( `year % 400 == 0` )

이 두 조건을 C/C++의 `if` 문으로 표현하면 다음과 같습니다.

```c
if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
    // 윤년일 때 (1 출력)
} else {
    // 윤년이 아닐 때 (0 출력)
}
