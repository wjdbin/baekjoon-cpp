# 백준 10172번: 개 (C++) 오답 노트

## ❓ 문제 상황
백준 10172번 C++ 코드를 제출했으나 '틀렸습니다' 판정을 받음.

## ❌ 원본 코드 (오류 원인)

```cpp
#include <iostream>
using namespace std;

int main()
{
	cout << "|\\_/|" << "\n";
	cout << "|q p|   /}" << "\n"; // 오류 1: '줄 바꿈 없는 공백( )'이 포함됨
	cout << "( 0 )\"\"\"\\" << "\n";
	cout << "|\" ^ \"`  |" << "\n"; // 오류 1, 2, 3
	cout << "||_/=\\\\__|" << "\n";

	return 0;
	
}
