# 📘 백준 1001번 — A-B (C++)

## 문제 요약
두 정수 `A`와 `B`가 주어졌을 때 **A - B**를 출력한다.

---

## 입출력 형식
**입력**: 한 줄에 정수 `A B`  
**출력**: 한 줄에 `A - B`

### 예제
```
입력: 3 2
출력: 1
```

---

## 풀이 아이디어
- 공백으로 구분된 두 정수를 입력받는다.
- 첫 번째 수에서 두 번째 수를 뺀 값을 그대로 출력한다.

---

## 주의할 점
- 입력은 **콤마(,)** 가 아니라 **연속 추출 연산자 `>>`** 로 받아야 한다.
  ```cpp
  // ❌ 잘못된 예
  std::cin >> a, b;   // a만 입력되고 b는 입력 안 됨(콤마 연산자)

  // ✅ 올바른 예
  std::cin >> a >> b;
  ```
- 출력에는 불필요한 문구 없이 결과만 출력한다(채점 시스템은 정확한 출력 형식을 기대함).
