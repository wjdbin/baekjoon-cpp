## 백준 10171 (고양이) C++ 풀이 핵심 개념

제출하신 코드는 C++의 기본 출력 기능과 **이스케이프 시퀀스(Escape Sequence)**를 정확히 이해하고 사용한 코드입니다.

---

### 1. C++ 기본 입출력

* **`#include <iostream>`**: C++의 표준 입출력(Input/Output Stream) 라이브러리를 포함합니다. `cout`을 사용하기 위해 필수입니다.
* **`using namespace std;`**: `cout`이나 `endl` 같은 표준 기능들을 `std::` 접두사 없이 사용할 수 있게 해줍니다.
* **`int main() { ... }`**: 프로그램이 시작되는 메인 함수입니다.
* **`cout << "..." << "\n";`**: `cout`은 콘솔에 문자열을 출력하는 객체입니다. `<<` 연산자를 사용해 출력할 내용을 전달합니다.
* **`\n`**: 'Newline'을 의미하는 이스케이프 시퀀스로, 줄바꿈을 수행합니다. (`endl`과 유사한 기능을 합니다.)

---

### 2. 이스케이프 시퀀스 (Escape Sequence)

이 문제의 핵심입니다. 문자열 안에서 특정 문자들은 특별한 기능을 수행하거나, 문법적으로 다른 의미를 가지기 때문에 그냥 사용할 수 없습니다. 이때 백슬래시(`\`)를 사용하여 해당 문자를 '탈출(escape)'시켜 원래의 문자 그대로 출력하게 만듭니다.

* **`\\`**
    * **이유**: 백슬래시(`\`)는 이스케이프 시퀀스를 시작하는 특수 문자입니다.
    * **사용**: `\` 문자 자체를 출력하려면 `\\`라고 두 번 입력해야 합니다.
    * **적용된 코드**: `cout << "\\    /\\" << "\n";`

* **`\'`**
    * **이유**: 작은따옴표(`'`)는 `char` (단일 문자) 자료형을 감쌀 때 사용됩니다. (예: `char c = 'A';`)
    * **사용**: 문자열(`"..."`) 안에서는 `'`를 그냥 써도 대부분의 컴파일러가 허용하지만, 문법적으로 명확하게 `'` 문자 자체임을 나타내기 위해 `\'`를 사용하는 것이 좋습니다.
    * **적용된 코드**: `cout << " )  ( ')" << "\n";`

* **(참고) `\"`**
    * **이유**: 큰따옴표(`"`)는 문자열(string)을 감싸는 데 사용됩니다.
    * **사용**: 문자열 안에서 `"` 문자 자체를 출력하려면 `\"`를 사용해야 합니다.
